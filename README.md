### Лабораторная работа №1. Основы обработки изображений
Цель: на практике закрепить полученные знания по базовым методам работы с изображениями.

Медианный фильтр представляет собой нелинейный метод цифровой обработки сигналов и изображений, основанный на ранговой статистике. В отличие от линейных фильтров, использующих свертку с фиксированным ядром, медианный фильтр заменяет значение центрального элемента скользящего окна на медиану значений всех элементов, попадающих в данную окрестность. Размер окна, как правило, выбирается нечетным для обеспечения симметричности относительно центрального пикселя.

Принцип работы медианного фильтра заключается в формировании упорядоченного ряда значений яркости пикселей в пределах апертуры фильтра с последующей заменой центрального отсчета на срединное значение этого вариационного ряда. Математически для одномерного случая выходной сигнал определяется как \( $y_i = \text{median}\{x_{i-k}, ..., x_i, ..., x_{i+k}\} \$
), где \( k \) определяет радиус окна фильтрации. Для двумерных изображений операция выполняется последовательно для каждого канала цветового пространства с учетом заданной маски окрестности.

Ключевым преимуществом медианной фильтрации является ее способность эффективно подавлять импульсные помехи (шум типа "соль и перец") при одновременном сохранении перепадов яркости и границ объектов. Это свойство обусловлено тем, что медиана, в отличие от среднего арифметического, нечувствительна к выбросам и экстремальным значениям, что позволяет удалять аномальные пиксели, не оказывая существенного влияния на структурные элементы изображения. Дополнительным достоинством метода выступает отсутствие сглаживания контуров, характерного для линейных низкочастотных фильтров.

К ограничениям медианного фильтра следует отнести вычислительную сложность, связанную с необходимостью сортировки элементов скользящего окна для каждого положения фильтра. При увеличении размера апертуры вычислительные затраты возрастают пропорционально квадрату линейных размеров окна. Кроме того, при обработке тонких линий и мелких деталей, размер которых меньше половины апертуры фильтра, возможно их частичное или полное подавление, что требует тщательного подбора параметров фильтрации в зависимости от характеристик обрабатываемого изображения и поставленных задач.
